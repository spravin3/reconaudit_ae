Idea B The Verification Loop
Turn 1: the LLM generates hypotheses and writes its own verification SQL for each one. Turns 2–N: every SQL runs against DuckDB before the LLM is allowed to issue a verdict. The final memo contains only what passed.

Idea D Single-Turn vs. Loop — What Survived Verification
The loop memo is shorter. That is not a failure — it is the loop working. Every finding it surfaces has a SELECT statement behind it. The CFO can trust what the loop says precisely because it will not say anything it has not checked.
